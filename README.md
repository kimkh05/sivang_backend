# Sivang

Sivang is an automatic household ledger.

This project was created in '프로젝트 실무II' at Daedeok Software Meister HighSchool.

## Using Skills
- Express
- TypeScript
- Prisma
- MySQL
- JWT

## Clone this repository

```shell
git clone https://github.com/kimkh05/sivang_backend.git [dir]
```

## Getting Start

```shell
yarn

yarn dev
```
